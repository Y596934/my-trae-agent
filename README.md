# My Trae AI Agent Project

## 项目简介
这是一个基于 **Trae IDE** 和 **DeepSeek API** 开发的自动化脚本生成 Agent 项目。

## 核心功能
- 通过 Trae IDE 的 SOLO 模式，用自然语言描述需求
- AI 自动完成：需求理解 → 任务拆解 → 代码生成 → 单元测试
- 用于自动化日志分析、服务器监控脚本生成等场景

## 实际使用数据
- **日均 API 调用**：约 40 次
- **月度总调用**：约 1,200 次
- **月度 Token 消耗**：约 64.31 元（基于 DeepSeek API）
